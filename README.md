## ✨ 插件简介

这是一款为 [AstrBot](https://github.com/Soulter/AstrBot) 打造的轻量化雀魂查谱与战绩吐槽插件。

它可以抓取玩家的真实战绩数据（一位率、放铳率等），或解析雀魂牌谱链接，并结合 AI 的人格设定（如傲娇毒舌教练、赛博算命大师）对玩家进行“惨无人道”的吐槽与点评！数据冰冷，但 AI 的嘴臭是有温度的！

## 🚀 指令与触发方式

当本插件启用后，用户即可通过自然语言或特定链接唤醒日麻教练：

| 触发方式 | 说明 |
| :--- | :--- |
| **聊天中提及** `"查雀魂 [昵称]"` | 触发 AI 工具，查询目标玩家详细战绩，并结合人格进行锐评。 |
| **发送牌谱链接** `https://game.maj...` | 触发牌谱解析工具，AI 将化身“赛博算命大师”对这局牌进行瞎编与吐槽。 |
| **纯文本指令** `/查雀魂 [昵称]` | 保底硬核纯数据模式，直接返回未加修饰的冰冷战绩数据。 |

## ⚙️ 豪华配置面板

本插件支持在 AstrBot 的 WebUI 管理后台进行可视化配置：

| 配置项 | 类型 | 默认值 | 说明 |
| :--- | :---: | :---: | :--- |
| **使用系统全局人格** | `开关` | 关 | 开启后将忽略下方选择，强制使用 AstrBot 当前的全局人格进行点评。 |
| **内置点评人格** | `下拉栏` | 傲娇毒舌 | 提供多种内置的日麻专属吐槽人格供随时切换。 |
| **毒舌火力值** | `数字` | 3 | 设定吐槽的猛烈程度 (1-3级)，3级为火力全开疯狂输出！ |

## 📦 安装方法

建议直接在 AstrBot 的 WebUI 插件市场中搜索 `astrbot_plugin_majsoul` 进行一键安装。
